## 🏃‍♀️ Run It Locally

Clone the project

```bash
$ git clone https://github.com/itchatapp/client.git
```

Go to the project directory

```bash
$ cd client
```

Install dependencies

```bash
$ npm install
```

then

```bash
$ npm run dev
```
